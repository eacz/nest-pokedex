# Setup for development

1. Clone the repository
2. Run
```
yarn install
```
3. Have installed Nest CLI
```
npm i -g @nestjs/cli
```
4. Setup database with docker
```
docker compose up -d
```

## Tech stack
* Nest 
* MongoDB
* Mongoose
